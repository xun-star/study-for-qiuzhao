# 有序数组的平方

[题目链接](https://leetcode.cn/problems/squares-of-a-sorted-array/description/)

>
>
>解题思路
>
>1.对数组中的每个元素平方
>
>2.排序数组

```c++
class Solution {
public:
    vector<int> sortedSquares(vector<int>& nums) {
        for(int i = 0;i<nums.size();i++){
            nums[i] = nums[i] * nums[i];
        }
        sort(nums.begin(),nums.end());
        return nums;

    }
};
```
